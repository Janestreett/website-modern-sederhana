## Deskripsi Proyek
Website ini merupakan **Demo Website** yang dibuat untuk memenuhi tugas Praktik 2.  
Website ini menampilkan tampilan modern, responsif, dan semua fitur berfungsi dengan baik saat dijalankan di browser.

## Fitur yang Tersedia
- Navbar navigasi yang lengkap
- Hero Section yang menarik
- Card Content (menampilkan proyek)
- Footer
- Desain responsif (ramah untuk HP / mobile)
- Tombol yang interaktif

## Teknologi yang Digunakan
- HTML5
- CSS3 (Internal Style)
- Responsive Design dengan Media Query

## Cara Menjalankan Website
1. Simpan kode sebagai file `index.html`
2. Buka file `index.html` menggunakan browser (Google Chrome / Microsoft Edge)
3. Website akan langsung berjalan
4. Semua fitur dapat diakses dan berfungsi
